Sobre

Site criado para a disciplina de Programação Web, com o intuito de apresentar um projeto de site de streaming de filmes e séries. O site é responsivo e utiliza Bootstrap para garantir uma boa experiência em dispositivos móveis e desktops. O site apresenta uma lista de filmes e séries, com informações detalhadas sobre cada item, incluindo título, diretor, elenco, data de lançamento, gênero, duração e sinopse. Além disso, o site conta com um carrossel de itens em destaque e uma seção de fotos do item.
